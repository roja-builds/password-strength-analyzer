# Password Strength Analyzer

## 📌 About
Password Strength Analyzer is a Python project that checks the strength of a password and suggests improvements.

## ✨ Features
- Checks password strength
- Verifies password length
- Checks uppercase and lowercase letters
- Checks numbers
- Checks special characters
- Suggests how to make the password stronger

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python
2. Run:
   python password_strength.py
